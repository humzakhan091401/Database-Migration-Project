# Database Migration & Normalization Project

## Overview

Developed a database migration and normalization solution using SQL and Oracle. The project focuses on relational database design, schema optimization, data migration, and maintaining data integrity.

## Technologies Used

* SQL
* Oracle SQL
* PL/SQL

## Key Features

* Designed and normalized relational database schemas.
* Created SQL scripts for table creation and data insertion.
* Implemented foreign key relationships and lookup tables.
* Developed PL/SQL procedures and triggers for automation.
* Performed data migration and transformation between database environments.
* Ensured data integrity through normalization and validation techniques.

## Learning Outcomes

* Relational Database Design
* Database Normalization
* Data Migration Techniques
* PL/SQL Development
* Data Integrity Management
